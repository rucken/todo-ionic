rucken-todo-ionic





Admin UI for Ionic4 with Angular7+ mobile todo application based on Rucken template

Usage

git clone https://github.com/rucken/todo-ionic.git my-app
cd my-app
npm install
npm run start:prod

Demo

https://rucken.github.io/todo-ionic/ - Static site on GitHub Pages

rucken-todo-ionic.apk - Mobile application for android

Compodoc documentations

http://rucken.github.io/todo-ionic/doc/demo - Demo application documentation
